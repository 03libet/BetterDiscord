BetterDiscord Documentation

Welcome to the BetterDiscord documentation. If you encounter any problems when using BetterDiscord, have a look at the troubleshooting guides, raise an issue on GitHub or join the Discord Server.

What is BetterDiscord?

BetterDiscord is in continuous development and new features get added daily! it is a HUGE enhancement to Discord. With custom Plugins and Themes you can have a Discordapp that is on steroids!

Uninstall

Having a bad day and want to uninstall BetterDiscord? Before you go on Please check our troubleshooting guides to help you in any way if not just sumbit an issue on our Github and we will get back to you shortly!

Windows

  • Remove the following 

    %appdata%\BetterDiscord
%localappdata%\Discord\app-x.x.xxx\resources\node_modules\BetterDiscord
%localappdata%\Discord\app-x.x.xxx\resources\app

Mac

  • Remove the following 

    ~/Library/Preferences/BetterDiscord
~/Library/Application Support/Discord/app-x.x.xxx/resources/node_modules/BetterDiscord
~/Library/Application Support/Discord/app-x.x.xxx/resources/app
